# Gambling Room

Simple PHP dice gambling game with a start page, gameplay logic, and results display.

## Features
- Enter player names
- Choose number of dice and throws
- View results and winner podium

## Project Structure
- index.php: Start page and form
- game.php: Game logic and session handling
- results.php: Results page
- styles/: Stylesheets
- images/: Images (dice and background)

## How to Run
### Option A: PHP built-in server
1. Start a local PHP server in the project folder:
	```bash
	php -S localhost:8000
	```
2. Open http://localhost:8000 in your browser.

### Option B: XAMPP on Windows
1. Copy the project folder into `C:\xampp\htdocs\gambling-room`.
2. Start Apache from the XAMPP Control Panel.
3. Open http://localhost/gambling-room in your browser.
